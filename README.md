# Basic Github API based app


## To run locally:

```terminal
$ git clone https://github.com/keshavga1209/github-clone.git
$ cd github-clone
$ npm install
$ npm start (to start the frontend)
$ Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
```
